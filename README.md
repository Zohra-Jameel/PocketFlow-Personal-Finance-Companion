
# PocketFlow – Personal Finance Companion

## 📱 Overview

PocketFlow is a lightweight personal finance mobile application designed to help users track daily transactions, monitor spending habits, and achieve simple financial goals.

The app focuses on clarity, ease of use, and meaningful insights, making it suitable for everyday financial tracking.

---

## ✨ Features

### 🏠 Dashboard

* Displays current balance, total income, and expenses
* Shows savings goal progress with a visual indicator
* Provides a quick overview of financial health

---

### 💰 Transaction Management

* Add transactions with amount, category, and type
* View transaction history
* Delete transactions
* Simple and intuitive data entry flow

---

### 🎯 Savings Goal

* Set and track a monthly savings goal
* Visual progress bar to monitor performance
* Encourages better saving habits

---

### 📊 Insights

* Displays total weekly spending
* Highlights highest spending category
* Helps users understand spending patterns

---

## 🛠️ Tech Stack

* Flutter
* Dart
* Local state management
* Simple in-memory data handling

---

##  Product Thinking

The app is designed as a daily-use companion rather than a complex financial system.

Design choices include:

* Minimal UI to reduce cognitive load
* Bottom navigation for intuitive access
* Fast transaction entry for regular usage
* Goal tracking to encourage behavioral change

---

## ⚙️ Setup Instructions

1. Clone repository
2. Run `flutter pub get`
3. Run `flutter run`

---

## 📌 Assumptions

* Data is stored locally
* Users manually enter transactions
* App is for personal use only
